# Bachelor Thesis Code Driven Infrastructure and Deployment

## Setup

* Install [Prince](http://www.princexml.com)
* `npm install`

## Optimize SVGs

```
./node_modules/.bin/svgo --pretty --folder images/
```

## Validate HTMl

```
./node_modules/.bin/html-validator --file=index.html
```

## Generate PDF

```
./pdf.sh
```
